PROJECT GOAL

我做的是什麼？

Ray's CPBL Data Site 是一個中華職棒非官方數據網站，目標不是只複製官網， 而是把賽程、比分、球隊、戰績、比賽中心與資料品質整合成一個更容易閱讀的資料系統。

即時比賽LIVE 比分、R/H/E、逐局、打者投手
球隊系統球隊總覽、名單、分析、異動
資料穩定dataQuality、finalLock、fallback、人工修正
02
WHY THIS SITE

跟一般比分網站差在哪？

一般比分網站

顯示結果，但不一定告訴你資料是否完整、是否正在同步、是否被鎖定。

Ray's CPBL Data Site

不只顯示資料，也判斷資料品質：confirmed、partial、debug、finalLock、recentFinalGuard。

04
LIVE SYSTEM

LIVE 比賽中心怎麼穩住？

Side Guard

避免客隊球員被錯塞到主隊，表格隊名不明確時不硬塞。

Stats Fallback

中文 boxscore 只吐單隊時，改由 stats fallback 補齊雙隊打者與投手。

Live State

開始解析目前局數、打者、投手、PITCH 與最近事件。

05
FINAL LOCK

FINAL 資料穩定化

賽後資料會寫入 finalLock，避免後續 LIVE 或 PREGAME 更新把終場資料污染。 近期 0～2 天 FINAL 場次會啟動 recentFinalGuard，強制交叉驗證，避免官方資料同步延遲造成錯誤。
